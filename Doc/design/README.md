## 核心设计变更（v3.0 vs v2.1）

### 移除的概念
- ❌ 周数概念（「第 X 周」）→ 改为游戏天数
- ❌ 阶段名称（破冰期/起步期/成长期/扩张期）→ 改为玩家自定义里程碑
- ❌ 行动点系统 → 移除，玩家自由行动

### 新增/重构的概念
- ✅ **游戏时间按天计算**（而非周）
- ✅ **玩家自设目标**：Onboarding 结束时玩家与 Lenny 共同制定 OPC 方向 + 里程碑
- ✅ **里程碑驱动进程**：游戏进度由玩家完成里程碑来推进，而非时间周期
- ✅ **自由探索阶段**：Onboarding 完成后完全开放，玩家自主决定节奏
- ✅ **Lenny Onboarding 流程**：收到邮件 → 王子墨推荐 → Safari 搜索 Lenny → X 私信 → 获得 TG → 与 Lenny 确定 OPC 方向 → 里程碑拆解 → 完成
